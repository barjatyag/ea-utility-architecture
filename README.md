## About This Repository

This repository contains a curated collection of **Enterprise Architecture artefacts** covering the Utility and Energy sector — built to ArchiMate® 3.1 and TOGAF ADM standards. All diagrams are standalone HTML files that render directly in any browser with no dependencies.

These artefacts are designed for:
- LinkedIn publishing & professional thought leadership
- Client-facing architecture discussions
- Reference architectures for utility digital transformation programmes

