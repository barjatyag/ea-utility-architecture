// IEC 62443 Security Controls — Zone & Conduit Requirements
🔐
Identity & Access Management
Role-based access control (RBAC); MFA for all OT remote access; PAM session recording; time-limited credentials
IEC 62443-3-3 SR 1.1–1.13 · NERC CIP-007
🧱
Zone & Conduit Segmentation
Firewall between every zone; no default-permit rules; conduit control with explicit allow-lists; VLAN per cell/area
IEC 62443-3-2 · NERC CIP-005
🔒
Data Confidentiality & Integrity
TLS 1.3 for all IT communications; OPC-UA certificate auth; IEC 61850 Ed2 security extensions; signed configs
IEC 62443-3-3 SR 4.1–4.2
👁
Continuous Monitoring (OT SOC)
Dragos / Claroty passive OT monitoring; ICS-specific threat signatures; Splunk SIEM; 24×7 SOC alert triage
IEC 62443-2-1 · NERC CIP-007 R4
🩹
Patch & Vulnerability Mgmt
Controlled OT patch delivery via DMZ proxy; vendor-tested patches only; compensating controls where unpatched
IEC 62443-2-3 · NERC CIP-007 R2
🔑
Cryptographic Key Management
HSM-backed PKI; X.509 device certs for IEDs/meters; automated certificate lifecycle; no shared credentials
IEC 62351 · IEC 62443-3-3 SR 1.8
🗂
OT Asset Inventory & Config Ctrl
Automated OT asset discovery (Claroty/Nozomi); configuration baseline; change management; software BOM
IEC 62443-2-1 · NERC CIP-010
🚨
Incident Response & Recovery
OT-specific IRP; backup offline media; tested recovery procedures; NERC CIP-009 compliant BCP/DR
