📐 Guiding Architecture Principles
P01
Open Standards First
IEC CIM, IEC 61850, IEC 62443, TOGAF — vendor lock-in minimised at every layer
P02
Secure by Design
Zero-trust architecture; IEC 62443 zone segmentation; security embedded not bolted on
P03
Data as a Strategic Asset
Single source of truth; master data governance; data quality SLAs enforced at ingestion
P04
Cloud-Native Where Appropriate
Cloud-first for IT systems; on-prem for latency-critical OT; hybrid for integration tier
P05
API-Led Connectivity
All integration via managed APIs; ESB as canonical broker; no point-to-point interfaces
P06
Composable Architecture
Loosely coupled microservices; domain-driven design; bounded contexts per utility domain
